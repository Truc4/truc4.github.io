StreamPlusBizhawk

StreamPlusBizhawk is an extension I made for Twitch that allows for interaction between stream viewers and the Bizhawk emulator. Using lua modules you can play games and viewers can manipulate elements of the game with this extension. It was created in four parts, the front end, which is a web application, the backend server made with NodeJS and hosted with AWS lambda, the broadcaster client, made with NodeJS, and the backend scripts, made with lua, all communicate together so the extension works.
